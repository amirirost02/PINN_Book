ЩҒШөЩ„ Ы№ ШӘШ®Щ…ЫҢЩҶ ШӯШ§Щ„ШӘ ЩҲ ШіЩҶШіЩҲШұ Щ…Ш¬Ш§ШІЫҢ ШЁШ§ PINN

ЩҒШөЩ„ Ы№: ШӘШ®Щ…ЫҢЩҶ ШӯШ§Щ„ШӘ ЩҲ ШіЩҶШіЩҲШұ Щ…Ш¬Ш§ШІЫҢ ШЁШ§ PINN

Ш§ЩҮШҜШ§ЩҒ ЫҢШ§ШҜЪҜЫҢШұЫҢ

ЩҫШі Ш§ШІ Щ…Ш·Ш§Щ„Ш№ЩҮЩ” Ш§ЫҢЩҶ ЩҒШөЩ„ШҢ ШҙЩ…Ш§ ЩӮШ§ШҜШұ Ш®ЩҲШ§ЩҮЫҢШҜ ШЁЩҲШҜ:

  1. Щ…ШҙЪ©Щ„ Ш§ЩҶШӯШұШ§ЩҒ (Drift) ШҜШұ Ш§ЩҶШҜШ§ШІЩҮвҖҢЪҜЫҢШұЫҢвҖҢЩҮШ§ЫҢ Ш§ШұШӘШ№Ш§ШҙЫҢ ШұШ§ ШӘЩҲШ¶ЫҢШӯ ШҜЩҮЫҢШҜ.
  2. ЫҢЪ© Щ…ШіШҰЩ„ЩҮЩ” ШӘШ®Щ…ЫҢЩҶ ШӯШ§Щ„ШӘ (State Estimation) ШұШ§ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ PINN ЩҒШұЩ…ЩҲЩ„вҖҢШЁЩҶШҜЫҢ Ъ©ЩҶЫҢШҜ.
  3. ШӘШ§ШЁШ№ ЩҮШІЫҢЩҶЩҮЩ” ШӘШұЪ©ЫҢШЁЫҢ ШҙШ§Щ…Щ„ ШӘЩ„ЩҒШ§ШӘ ШҜШ§ШҜЩҮ (ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШҙШӘШ§ШЁ ЩҶЩҲЫҢШІЫҢ)ШҢ ШӘЩ„ЩҒШ§ШӘ ЩҒЫҢШІЫҢЪ© (Щ…Ш№Ш§ШҜЩ„ЩҮ ШӯШұЪ©ШӘ)ШҢ ЩҲ ШӘЩ„ЩҒШ§ШӘ ШҙШұШ§ЫҢШ· Ш§ЩҲЩ„ЫҢЩҮ ШұШ§ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ъ©ЩҶЫҢШҜ.
  4. ЫҢЪ© PINN ШұШ§ ШЁШұШ§ЫҢ ШӘШ®Щ…ЫҢЩҶ Ш¬Ш§ШЁЩҮвҖҢШ¬Ш§ЫҢЫҢ ЩҲ ШіШұШ№ШӘ Ш§ШІ ШұЩҲЫҢ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШҙШӘШ§ШЁ ЩҶЩҲЫҢШІЫҢ Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ.
  5. Ш№Щ…Щ„Ъ©ШұШҜ PINN ШұШ§ ШЁШ§ ШұЩҲШҙ Ш§ЩҶШӘЪҜШұШ§Щ„вҖҢЪҜЫҢШұЫҢ Ш№ШҜШҜЫҢ ШіЩҶШӘЫҢ Щ…ЩӮШ§ЫҢШіЩҮ Ъ©ШұШҜЩҮ ЩҲ Щ…ШІШ§ЫҢШ§ ЩҲ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢ ШўЩҶ ШұШ§ ШӘШӯЩ„ЫҢЩ„ ЩҶЩ…Ш§ЫҢЫҢШҜ.

ЪҶШ§Щ„Шҙ ШӘШ®Щ…ЫҢЩҶ ШӯШ§Щ„ШӘ ШҜШұ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ Ш§ШұШӘШ№Ш§ШҙЫҢ

ШҜШұ ШЁШіЫҢШ§ШұЫҢ Ш§ШІ Ъ©Ш§ШұШЁШұШҜЩҮШ§ЫҢ Щ…ЩҮЩҶШҜШіЫҢ вҖ” ШЁЩҮвҖҢЩҲЫҢЪҳЩҮ ШҜШұ ЩҫШ§ЫҢШҙ ШіЩ„Ш§Щ…ШӘ ШіШ§ШІЩҮвҖҢЩҮШ§ШҢ ШӘШӯЩ„ЫҢЩ„ Ш§ШұШӘШ№Ш§ШҙШ§ШӘ ШЁЩҲЪҳЫҢвҖҢЩҮШ§ЫҢ ШұЫҢЩ„ЫҢШҢ ЩҲ Ъ©ЩҶШӘШұЩ„ ЩҒШ№Ш§Щ„ вҖ” ШӘЩҶЩҮШ§ Ъ©Щ…ЫҢШӘЫҢ Ъ©ЩҮ Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ Ш§ЩҶШҜШ§ШІЩҮвҖҢЪҜЫҢШұЫҢ Щ…ЫҢвҖҢШҙЩҲШҜШҢ ШҙШӘШ§ШЁ Ш§ШіШӘ. ШҙШӘШ§ШЁвҖҢШіЩҶШ¬вҖҢЩҮШ§ Ш§ШұШІШ§ЩҶШҢ Ъ©ЩҲЪҶЪ© ЩҲ ЩӮШ§ШЁЩ„вҖҢШ§Ш·Щ…ЫҢЩҶШ§ЩҶ ЩҮШіШӘЩҶШҜШҢ Ш§Щ…Ш§ ШЁШұШ§ЫҢ ШӘШӯЩ„ЫҢЩ„ ШҜЩӮЫҢЩӮвҖҢШӘШұШҢ Щ…ЩҮЩҶШҜШі ШЁЩҮ Ш¬Ш§ШЁЩҮвҖҢШ¬Ш§ЫҢЫҢ ЩҲ ШіШұШ№ШӘ ЩҶЫҢШІ ЩҶЫҢШ§ШІ ШҜШ§ШұШҜ.

ШұЩҲШҙ Ъ©Щ„Ш§ШіЫҢЪ©: Ш§ЩҶШӘЪҜШұШ§Щ„вҖҢЪҜЫҢШұЫҢ Ш№ШҜШҜЫҢ
ШіШ§ШҜЩҮвҖҢШӘШұЫҢЩҶ ШұШ§ЩҮ ШЁШұШ§ЫҢ Щ…ШӯШ§ШіШЁЩҮЩ” Ш¬Ш§ШЁЩҮвҖҢШ¬Ш§ЫҢЫҢ Ш§ШІ ШұЩҲЫҢ ШҙШӘШ§ШЁШҢ ШҜЩҲ ШЁШ§Шұ Ш§ЩҶШӘЪҜШұШ§Щ„вҖҢЪҜЫҢШұЫҢ Ш№ШҜШҜЫҢ Ш§ШіШӘ. Ш§Щ…Ш§ Ш§ЫҢЩҶ ШұЩҲШҙ ШЁШ§ ЫҢЪ© ЩҶЩӮШө ШЁЩҶЫҢШ§ШҜЫҢЩҶ Щ…ЩҲШ§Ш¬ЩҮ Ш§ШіШӘ: Ш§ЩҶШӯШұШ§ЩҒ (Drift). ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШҙШӘШ§ШЁ ЩҮЩ…ЩҲШ§ШұЩҮ ШҜШ§ШұШ§ЫҢ ЩҶЩҲЫҢШІ ЩҒШұЪ©Ш§ЩҶШі ЩҫШ§ЫҢЫҢЩҶ ЩҮШіШӘЩҶШҜ Ъ©ЩҮ ШҜШұ Ш§ЩҶШӘЪҜШұШ§Щ„вҖҢЪҜЫҢШұЫҢ ШЁЩҮ Ш¬Ш§ШЁЩҮвҖҢШ¬Ш§ЫҢЫҢ ШӘШЁШҜЫҢЩ„ ШҙШҜЩҮ ЩҲ ШЁШ§Ш№Ш« Ш§ЩҶШӯШұШ§ЩҒ ШҙШҜЫҢШҜ Щ…ЫҢвҖҢШҙЩҲШҜ.

ШұЩҲШҙвҖҢЩҮШ§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮвҖҢШӘШұ (ЩҒЫҢЩ„ШӘШұ Ъ©Ш§Щ„Щ…ЩҶ)
ЩҒЫҢЩ„ШӘШұ Ъ©Ш§Щ„Щ…ЩҶ ЫҢЪ© ШұЩҲШҙ ШЁЩҮЫҢЩҶЩҮ ШЁШұШ§ЫҢ ШӘШ®Щ…ЫҢЩҶ ШӯШ§Щ„ШӘ Ш§ШіШӘШҢ ШЁЩҮвҖҢШҙШұШ·ЫҢ Ъ©ЩҮ Щ…ШҜЩ„ ШҜЩӮЫҢЩӮ ШіЫҢШіШӘЩ… ЩҲ Ъ©ЩҲЩҲШ§ШұЫҢШ§ЩҶШі ЩҶЩҲЫҢШІЩҮШ§ ШҙЩҶШ§Ш®ШӘЩҮ ШҙШҜЩҮ ШЁШ§ШҙЩҶШҜ. ШҜШұ Ш№Щ…Щ„ШҢ ШӘЩҶШёЫҢЩ… Щ…Ш§ШӘШұЫҢШівҖҢЩҮШ§ЫҢ Ъ©ЩҲЩҲШ§ШұЫҢШ§ЩҶШі ЩҶЫҢШ§ШІЩ…ЩҶШҜ ШҜШ§ЩҶШҙ ЩӮШЁЩ„ЫҢ ЩҲ ШўШІЩ…ЩҲЩҶ ЩҲ Ш®Ш·Ш§ Ш§ШіШӘ.

ШұШ§ЩҮвҖҢШӯЩ„ ЩҫЫҢШҙЩҶЩҮШ§ШҜЫҢ: PINN ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ ЫҢЪ© ЩҒЫҢЩ„ШӘШұ ЩҮЩ…ЩҲШ§ШұЪ©ЩҶЩҶШҜЩҮ
PINN ШЁШ§ ШӘШұЪ©ЫҢШЁ ШҜЩҲ Щ…ЩҶШЁШ№ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ (ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШҙШӘШ§ШЁ ЩҶЩҲЫҢШІЫҢ ЩҲ Щ…Ш№Ш§ШҜЩ„ЩҮЩ” ШӯШұЪ©ШӘ) Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ Ш¬Ш§ШЁЩҮвҖҢШ¬Ш§ЫҢЫҢ ЩҲ ШіШұШ№ШӘ ШұШ§ ШЁЩҮвҖҢЪҜЩҲЩҶЩҮвҖҢШ§ЫҢ ШЁШ§ШІШіШ§ШІЫҢ Ъ©ЩҶШҜ Ъ©ЩҮ ЩҮЩ… ШЁШ§ ШҜШ§ШҜЩҮвҖҢЩҮШ§ Щ…Ш·Ш§ШЁЩӮШӘ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ ЩҲ ЩҮЩ… ЩӮЩҲШ§ЩҶЫҢЩҶ ЩҒЫҢШІЫҢЪ© ШұШ§ ШұШ№Ш§ЫҢШӘ Ъ©ЩҶШҜ.

ШҙЪ©Щ„ Ы№-Ыұ: Щ…ЩӮШ§ЫҢШіЩҮЩ” Ш¬Ш§ШЁЩҮвҖҢШ¬Ш§ЫҢЫҢ ШӘШ®Щ…ЫҢЩҶвҖҢШІШҜЩҮвҖҢШҙШҜЩҮ ШЁШ§ PINN (Ш®Ш· ЩӮШұЩ…ШІ ЪҶЫҢЩҶ) ЩҲ Ш§ЩҶШӘЪҜШұШ§Щ„вҖҢЪҜЫҢШұЫҢ Ш№ШҜШҜЫҢ (Ш®Ш· ШіШЁШІ ЩҶЩӮШ·ЩҮвҖҢЪҶЫҢЩҶ) ШҜШұ ШЁШұШ§ШЁШұ Ш¬ЩҲШ§ШЁ ШӘШӯЩ„ЫҢЩ„ЫҢ (Ш®Ш· ШўШЁЫҢ Щ…Щ…ШӘШҜ)

ЩҒШұЩ…ЩҲЩ„вҖҢШЁЩҶШҜЫҢ ШұЫҢШ§Ш¶ЫҢ Щ…ШіШҰЩ„ЩҮ

ШіЫҢШіШӘЩ… Ш¬ШұЩ…-ЩҒЩҶШұ-ШҜЩ…ЩҫШұ Ш®Ш·ЫҢ (Щ…ШҜЩ„ ШіШ§ШҜЩҮвҖҢШҙШҜЩҮЩ” ЫҢЪ© ШҜШұШ¬ЩҮЩ” ШўШІШ§ШҜЫҢ ШЁШұШ§ЫҢ ШЁЩҲЪҳЫҢ):

$$ m\ddot{x} + c\dot{x} + kx = 0, \quad x(0) = x_0, \quad \dot{x}(0) = v_0 $$

ЩҒШұШ¶ЫҢШ§ШӘ Щ…ШіШҰЩ„ЩҮ:

  • ЩҫШ§ШұШ§Щ…ШӘШұЩҮШ§ЫҢ ШіЫҢШіШӘЩ… ($m, c, k$) Ъ©Ш§Щ…Щ„Ш§ЩӢ ШҙЩҶШ§Ш®ШӘЩҮвҖҢШҙШҜЩҮ ЩҮШіШӘЩҶШҜ.
  • ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШҙШӘШ§ШЁ $\ddot{x}(t)$ ШҜШұ $N_{data}$ ЩҶЩӮШ·ЩҮЩ” ШІЩ…Ш§ЩҶЫҢ ШӘШөШ§ШҜЩҒЫҢ ШҜШұ ШЁШ§ШІЩҮЩ” $t \in [0, T]$ ШҜШұ ШҜШіШӘШұШі ЩҮШіШӘЩҶШҜ.
  • ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШҙШӘШ§ШЁ ЩҮЩ…ШұШ§ЩҮ ШЁШ§ ЩҶЩҲЫҢШІ ЪҜШ§ЩҲШіЫҢ ЩҮШіШӘЩҶШҜ.
  • ШҙШұШ§ЫҢШ· Ш§ЩҲЩ„ЫҢЩҮ ($x_0, v_0$) ШҙЩҶШ§Ш®ШӘЩҮвҖҢШҙШҜЩҮ ЩҮШіШӘЩҶШҜ.

ШӘШ№ШұЫҢЩҒ Щ…ШӘШәЫҢШұЩҮШ§ ШҜШұ PINN:

  • ЩҲШұЩҲШҜЫҢ ШҙШЁЪ©ЩҮ: $t$ (ШІЩ…Ш§ЩҶ)
  • Ш®ШұЩҲШ¬ЫҢ ШҙШЁЪ©ЩҮ: $x_{NN}(t)$ (Ш¬Ш§ШЁЩҮвҖҢШ¬Ш§ЫҢЫҢ)
  • Щ…ШҙШӘЩӮШ§ШӘ ШЁШ§ Autograd: ШіШұШ№ШӘ $\dot{x}_{NN}$ ЩҲ ШҙШӘШ§ШЁ $\ddot{x}_{NN}$ ШЁШ§ Щ…ШҙШӘЩӮвҖҢЪҜЫҢШұЫҢ Ш®ЩҲШҜЪ©Ш§Шұ Щ…ШӯШ§ШіШЁЩҮ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ.
ШҙЪ©Щ„ Ы№-ЫІ: Щ…ЩӮШ§ЫҢШіЩҮЩ” ШіШұШ№ШӘ ШӘШ®Щ…ЫҢЩҶвҖҢШІШҜЩҮвҖҢШҙШҜЩҮ ШЁШ§ PINN (Ш®Ш· ЩӮШұЩ…ШІ ЪҶЫҢЩҶ) ЩҲ Ш¬ЩҲШ§ШЁ ШӘШӯЩ„ЫҢЩ„ЫҢ (Ш®Ш· ШўШЁЫҢ Щ…Щ…ШӘШҜ)

ШӘШ§ШЁШ№ ЩҮШІЫҢЩҶЩҮ ШӘШұЪ©ЫҢШЁЫҢ

ШӘЩ„ЩҒШ§ШӘ ШҜШ§ШҜЩҮ (Щ…ЩӮШ§ЫҢШіЩҮ ШЁШ§ ШҙШӘШ§ШЁ ЩҶЩҲЫҢШІЫҢ)

$$ \mathcal{L}_{data} = \frac{1}{N_{data}} \sum_{i=1}^{N_{data}} \left( \ddot{x}_{NN}(t_i) - \ddot{x}_{measured}(t_i) \right)^2 $$

ШӘЩ„ЩҒШ§ШӘ ЩҒЫҢШІЫҢЪ© (Щ…Ш№Ш§ШҜЩ„ЩҮ ШӯШұЪ©ШӘ ШҜШұ ЩҶЩӮШ§Ш· Ъ©Ш§Щ„ЩҲЪ©ЫҢШҙЩҶ)

$$ \mathcal{L}_{physics} = \frac{1}{N_{col}} \sum_{j=1}^{N_{col}} \left( m \ddot{x}_{NN}(t_j) + c \dot{x}_{NN}(t_j) + k x_{NN}(t_j) \right)^2 $$

ШӘЩ„ЩҒШ§ШӘ ШҙШұШ§ЫҢШ· Ш§ЩҲЩ„ЫҢЩҮ

$$ \mathcal{L}_{ic} = \left( x_{NN}(0) - x_0 \right)^2 + \left( \dot{x}_{NN}(0) - v_0 \right)^2 $$

ШӘЩ„ЩҒШ§ШӘ Ъ©Щ„

$$ \mathcal{L}_{total} = \lambda_{data} \cdot \mathcal{L}_{data} + \lambda_{physics} \cdot \mathcal{L}_{physics} + \lambda_{ic} \cdot \mathcal{L}_{ic} $$

Ъ©ШҜ Ъ©Ш§Щ…Щ„ PINN ШЁШұШ§ЫҢ ШӘШ®Щ…ЫҢЩҶ ШӯШ§Щ„ШӘ

layers=[1, 50, 50, 50, 1]): super(PINN_StateEstimation, self).__init__() self.net = nn.Sequential() for i in range(len(layers) - 1): self.net.add_module(f'linear_{i}', nn.Linear(layers[i], layers[i+1])) if i < len(layers) - 2: self.net.add_module(f'tanh_{i}', nn.Tanh()) def forward(self, t): return self.net(t) # = 0.5 k_true = 10.0 x0_true = 1.0 v0_true = 0.0 T = 10.0 omega_n = np.sqrt(k_true / m_true) zeta = c_true / (2 * np.sqrt(k_true * m_true)) omega_d = omega_n * np.sqrt(1 - zeta**2) A = x0_true B = (v0_true + zeta * omega_n * x0_true) / omega_d def analytical_solution(t): return np.exp(-zeta * omega_n * t) * (A * np.cos(omega_d * t) + B * np.sin(omega_d * t)) def analytical_velocity(t): x = analytical_solution(t) v = -zeta * omega_n * x + np.exp(-zeta * omega_n * t) * ( -omega_d * A * np.sin(omega_d * t) + omega_d * B * np.cos(omega_d * t) ) return v def analytical_acceleration(t): x = analytical_solution(t) v = analytical_velocity(t) return -zeta * omega_n * v - omega_n**2 * x N_data = 200 t_data_np = np.random.rand(N_data) * T a_true = analytical_acceleration(t_data_np) noise_level = 0.05 noise = noise_level * np.std(a_true) * np.random.randn(N_data) a_noisy = a_true + noise t_data = torch.tensor(t_data_np, dtype=torch.float64, requires_grad=True).reshape(-1, 1) a_noisy = torch.tensor(a_noisy, dtype=torch.float64).reshape(-1, 1) # torch.rand(N_col, 1, requires_grad=True) * T model = PINN_StateEstimation() optimizer = torch.optim.Adam(model.parameters(), lr=1e-3) def compute_loss(): x_data = model(t_data) dx_data = torch.autograd.grad(x_data, t_data, grad_outputs=torch.ones_like(x_data), create_graph=True, retain_graph=True)[0] d2x_data = torch.autograd.grad(dx_data, t_data, grad_outputs=torch.ones_like(dx_data), create_graph=True, retain_graph=True)[0] loss_data = torch.mean((d2x_data - a_noisy)**2) x_col = model(t_col) dx_col = torch.autograd.grad(x_col, t_col, grad_outputs=torch.ones_like(x_col), create_graph=True, retain_graph=True)[0] d2x_col = torch.autograd.grad(dx_col, t_col, grad_outputs=torch.ones_like(dx_col), create_graph=True, retain_graph=True)[0] ode_residual = m_true * d2x_col + c_true * dx_col + k_true * x_col loss_physics = torch.mean(ode_residual**2) t_ic = torch.tensor([[0.0]], requires_grad=True) x_ic = model(t_ic) dx_ic = torch.autograd.grad(x_ic, t_ic, grad_outputs=torch.ones_like(x_ic), create_graph=True, retain_graph=True)[0] loss_ic = (x_ic - x0_true)**2 + (dx_ic - v0_true)**2 loss = loss_data + 1.0 * loss_physics + 10.0 * loss_ic return loss, loss_data, loss_physics, loss_ic # ШӘЩҶШёЫҢЩ… Ш§ШЁШұЩҫШ§ШұШ§Щ…ШӘШұЩҮШ§ЫҢ ШӘШ§ШЁШ№ ЩҮШІЫҢЩҶЩҮ

Ш§ШЁШұЩҫШ§ШұШ§Щ…ШӘШұ Щ…ЩӮШҜШ§Шұ ШҜШұ Ъ©ШҜ ЩҶЩӮШҙ ЪҶЩҮ ШІЩ…Ш§ЩҶЫҢ Ш§ЩҒШІШ§ЫҢШҙ ШҜЩҮЫҢШҜШҹ
$\lambda_{data}$ Ыұ.Ы° (Ш¶Щ…ЩҶЫҢ) Ш§ЩҮЩ…ЫҢШӘ ШӘШ·Ш§ШЁЩӮ ШЁШ§ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШҙШӘШ§ШЁ ЩҶЩҲЫҢШІЫҢ Ш§ЪҜШұ ШҙШЁЪ©ЩҮ ШҜШ§ШҜЩҮвҖҢЩҮШ§ ШұШ§ ЩҶШ§ШҜЫҢШҜЩҮ Щ…ЫҢвҖҢЪҜЫҢШұШҜ
$\lambda_{physics}$ Ыұ.Ы° Ш§ЩҮЩ…ЫҢШӘ Ш§ШұШ¶Ш§ЫҢ Щ…Ш№Ш§ШҜЩ„ЩҮ ШӯШұЪ©ШӘ Ш§ЪҜШұ ЩҫШ§ШіШ® ШҙШЁЪ©ЩҮ ШәЫҢШұЩҒЫҢШІЫҢЪ©ЫҢ Ш§ШіШӘ
$\lambda_{ic}$ ЫұЫ°.Ы° Ш§ЩҮЩ…ЫҢШӘ ШҙШұШ§ЫҢШ· Ш§ЩҲЩ„ЫҢЩҮ Ш§ЪҜШұ ШҙШұШ§ЫҢШ· Ш§ЩҲЩ„ЫҢЩҮ ЩҶЩӮШ¶ Щ…ЫҢвҖҢШҙЩҲШҜ
ШҙЪ©Щ„ Ы№-Ыі: ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШҙШӘШ§ШЁ ЩҶЩҲЫҢШІЫҢ (ЩҶЩӮШ§Ш· Ш®Ш§Ъ©ШіШӘШұЫҢ)ШҢ Ш¬ЩҲШ§ШЁ ШӘШӯЩ„ЫҢЩ„ЫҢ (Ш®Ш· ШўШЁЫҢ Щ…Щ…ШӘШҜ)ШҢ ЩҲ ШҙШӘШ§ШЁ ЩҮЩ…ЩҲШ§ШұШҙШҜЩҮ ШЁШ§ PINN (Ш®Ш· ЩӮШұЩ…ШІ ЪҶЫҢЩҶ)

ШӘШӯЩ„ЫҢЩ„ ЩҶШӘШ§ЫҢШ¬

ЩҫШі Ш§ШІ Ш§Ш¬ШұШ§ЫҢ Ъ©ШҜШҢ ЩҶШӘШ§ЫҢШ¬ ШІЫҢШұ ЩӮШ§ШЁЩ„вҖҢЩ…ШҙШ§ЩҮШҜЩҮ Ш§ШіШӘ:

  1. Ш¬Ш§ШЁЩҮвҖҢШ¬Ш§ЫҢЫҢ: ЩҫШ§ШіШ® PINN ШӘЩӮШұЫҢШЁШ§ЩӢ ШЁШұ ШұЩҲЫҢ Ш¬ЩҲШ§ШЁ ШӘШӯЩ„ЫҢЩ„ЫҢ Щ…ЩҶШ·ШЁЩӮ Ш§ШіШӘ. Ш®Ш·Ш§ЫҢ L2 ШӯШҜЩҲШҜ $10^{-4}$ ШӘШ§ $10^{-5}$ Ш§ШіШӘ. ШҜШұ Щ…ЩӮШ§ШЁЩ„ШҢ Ш§ЩҶШӘЪҜШұШ§Щ„вҖҢЪҜЫҢШұЫҢ Ш№ШҜШҜЫҢ ШЁЩҮвҖҢШҙШҜШӘ ШҜЪҶШ§Шұ Ш§ЩҶШӯШұШ§ЩҒ ШҙШҜЩҮ Ш§ШіШӘ.
  2. ШіШұШ№ШӘ: ШіШұШ№ШӘ ШӘШ®Щ…ЫҢЩҶвҖҢШІШҜЩҮвҖҢШҙШҜЩҮ ШӘЩҲШіШ· PINN ЩҶЫҢШІ ШӘШ·Ш§ШЁЩӮ Ш№Ш§Щ„ЫҢ ШЁШ§ Ш¬ЩҲШ§ШЁ ШӘШӯЩ„ЫҢЩ„ЫҢ ШҜШ§ШұШҜ.
  3. ШҙШӘШ§ШЁ: PINN ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩӮЫҢШҜ ЩҒЫҢШІЫҢЪ©ШҢ ЩҶЩҲЫҢШІ ШұШ§ ЩҒЫҢЩ„ШӘШұ Ъ©ШұШҜЩҮ ЩҲ ШҙШӘШ§ШЁ ЩҮЩ…ЩҲШ§Шұ ЩҲ ШҜЩӮЫҢЩӮЫҢ ШӘЩҲЩ„ЫҢШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

Щ…ШІШ§ЫҢШ§ ЩҲ Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§ЫҢ ШұЩҲШҙ

Щ…ШІШ§ЫҢШ§ ЩҶШіШЁШӘ ШЁЩҮ ШұЩҲШҙвҖҢЩҮШ§ЫҢ Ъ©Щ„Ш§ШіЫҢЪ©:

  1. Ш№ШҜЩ… Ш§ЩҶШӯШұШ§ЩҒ (Drift): PINN ШЁШ§ Ш§Ш№Щ…Ш§Щ„ Щ…Ш№Ш§ШҜЩ„ЩҮЩ” ШӯШұЪ©ШӘ ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ ЩӮЫҢШҜ ШіШ®ШӘШҢ ЩҫШ§ШіШ® ШұШ§ ШҜШұ Щ…ШіЫҢШұ ЩҒЫҢШІЫҢЪ©ЫҢ ЩҶЪҜЩҮ Щ…ЫҢвҖҢШҜШ§ШұШҜ.
  2. ЩҒЫҢЩ„ШӘШұ Ъ©ШұШҜЩҶ ЩҶЩҲЫҢШІ: PINN ЩҶЩҲЫҢШІ ШұШ§ ШЁШ§ ЩӮЫҢШҜ ЩҒЫҢШІЫҢЪ© ШӯШ°ЩҒ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ ЩҫШ§ШіШ® ЩҮЩ…ЩҲШ§Шұ ШӘЩҲЩ„ЫҢШҜ Щ…ЫҢвҖҢЪ©ЩҶШҜ.
  3. Ш§ШіШӘЩӮЩ„Ш§Щ„ Ш§ШІ Ъ©ЩҲЩҲШ§ШұЫҢШ§ЩҶШі ЩҶЩҲЫҢШІ: ШЁШұШ®Щ„Ш§ЩҒ ЩҒЫҢЩ„ШӘШұ Ъ©Ш§Щ„Щ…ЩҶШҢ PINN ШЁЩҮ Щ…ШҜЩ„ ЩҶЩҲЫҢШІ ЩҶЫҢШ§ШІ ЩҶШҜШ§ШұШҜ.

Щ…ШӯШҜЩҲШҜЫҢШӘвҖҢЩҮШ§ ЩҲ ЪҶШ§Щ„ШҙвҖҢЩҮШ§:

  1. ЩҶЫҢШ§ШІ ШЁЩҮ Щ…ШҜЩ„ ШҜЩӮЫҢЩӮ: Ш§ЪҜШұ Щ…Ш№Ш§ШҜЩ„ЩҮЩ” ШӯШұЪ©ШӘ ШҜЩӮЫҢЩӮ ЩҶШЁШ§ШҙШҜШҢ ШӘШ®Щ…ЫҢЩҶвҖҢЩҮШ§ ШЁШ§ Ш®Ш·Ш§ ЩҮЩ…ШұШ§ЩҮ Ш®ЩҲШ§ЩҮЩҶШҜ ШЁЩҲШҜ.
  2. ШӯШіШ§ШіЫҢШӘ ШЁЩҮ ЩҲШІЩҶвҖҢЩҮШ§ЫҢ ШӘШ§ШЁШ№ ЩҮШІЫҢЩҶЩҮ: Ш§ЩҶШӘШ®Ш§ШЁ ЩҶШ§Щ…ЩҶШ§ШіШЁ ЩҲШІЩҶвҖҢЩҮШ§ Щ…Щ…Ъ©ЩҶ Ш§ШіШӘ ШЁЩҮ Ш§ЩҲШұЩҒЫҢШӘЫҢЩҶЪҜ ЫҢШ§ ЩҶЩӮШ¶ ЩӮЫҢШҜ ЩҒЫҢШІЫҢЪ© Щ…ЩҶШ¬Шұ ШҙЩҲШҜ.
  3. ЩҮШІЫҢЩҶЩҮЩ” Щ…ШӯШ§ШіШЁШ§ШӘЫҢ: ШўЩ…ЩҲШІШҙ ШҙШЁЪ©ЩҮ ШЁШұШ§ЫҢ Ъ©Ш§ШұШЁШұШҜЩҮШ§ЫҢ ШЁШұШ®Ш· Щ…ЩҶШ§ШіШЁ ЩҶЫҢШіШӘ.

Ш¬Щ…Ш№вҖҢШЁЩҶШҜЫҢ ЩҲ Щ…ШіЫҢШұ ЩҫЫҢШҙвҖҢШұЩҲ

ШҜШұ Ш§ЫҢЩҶ ЩҒШөЩ„ШҢ ШЁШ§ ЫҢЪ©ЫҢ Ш§ШІ Ъ©Ш§ШұШЁШұШҜЫҢвҖҢШӘШұЫҢЩҶ Ъ©Ш§ШұШЁШұШҜЩҮШ§ЫҢ PINN ШҜШұ ШӯЩҲШІЩҮЩ” Ш§ШұШӘШ№Ш§ШҙШ§ШӘ ШўШҙЩҶШ§ ШҙШҜЫҢЩ…. ЩҶЪ©Ш§ШӘ Ъ©Щ„ЫҢШҜЫҢ:

  1. Щ…ШҙЪ©Щ„ Ш§ЩҶШӯШұШ§ЩҒ: Ш§ЩҶШӘЪҜШұШ§Щ„вҖҢЪҜЫҢШұЫҢ Ш№ШҜШҜЫҢ Ш§ШІ ШҜШ§ШҜЩҮвҖҢЩҮШ§ЫҢ ШҙШӘШ§ШЁ ЩҶЩҲЫҢШІЫҢ Щ…ЩҶШ¬Шұ ШЁЩҮ Ш§ЩҶШӯШұШ§ЩҒ ШҙШҜЫҢШҜ ШҜШұ Ш¬Ш§ШЁЩҮвҖҢШ¬Ш§ЫҢЫҢ Щ…ЫҢвҖҢШҙЩҲШҜ.
  2. ШұЩҲШҙ PINN: ШЁШ§ ШӘШ№ШұЫҢЩҒ Ш®ШұЩҲШ¬ЫҢ ШҙШЁЪ©ЩҮ ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ Ш¬Ш§ШЁЩҮвҖҢШ¬Ш§ЫҢЫҢ ЩҲ Щ…ШӯШ§ШіШЁЩҮЩ” ШҙШӘШ§ШЁ ШЁШ§ AutogradШҢ ШҙШЁЪ©ЩҮ ЩҮЩ…вҖҢШІЩ…Ш§ЩҶ ШЁШ§ ШҜШ§ШҜЩҮвҖҢЩҮШ§ ЩҲ Щ…Ш№Ш§ШҜЩ„ЩҮЩ” ШӯШұЪ©ШӘ ШўЩ…ЩҲШІШҙ Щ…ЫҢвҖҢШЁЫҢЩҶШҜ.
  3. ШӘШ§ШЁШ№ ЩҮШІЫҢЩҶЩҮ ШӘШұЪ©ЫҢШЁЫҢ: ШҙШ§Щ…Щ„ ШӘЩ„ЩҒШ§ШӘ ШҜШ§ШҜЩҮШҢ ШӘЩ„ЩҒШ§ШӘ ЩҒЫҢШІЫҢЪ©ШҢ ЩҲ ШӘЩ„ЩҒШ§ШӘ ШҙШұШ§ЫҢШ· Ш§ЩҲЩ„ЫҢЩҮ Ш§ШіШӘ.
  4. ЩҶШӘШ§ЫҢШ¬: PINN Ш¬Ш§ШЁЩҮвҖҢШ¬Ш§ЫҢЫҢ ЩҲ ШіШұШ№ШӘ ШұШ§ ШЁШҜЩҲЩҶ Ш§ЩҶШӯШұШ§ЩҒ ЩҲ ШЁШ§ ШҜЩӮШӘ ШЁШ§Щ„Ш§ ШӘШ®Щ…ЫҢЩҶ Щ…ЫҢвҖҢШІЩҶШҜ.

ЩҫЩ„ ШЁЩҮ ЩҒШөЩ„ ШҜЩҮЩ…: ШҜШұ ЩҒШөЩ„ ШЁШ№ШҜШҢ ШЁШ§ ЪҶШ§Щ„ШҙвҖҢЩҮШ§ЫҢ Ш№Щ…Щ„ЫҢ ШўЩ…ЩҲШІШҙ PINN (ЩҲШ§ЪҜШұШ§ЫҢЫҢШҢ ЩҶШ§ЩҫШ§ЫҢШҜШ§ШұЫҢШҢ ЩҲ ШІЩ…Ш§ЩҶвҖҢШЁШұЫҢ) ШўШҙЩҶШ§ Ш®ЩҲШ§ЩҮЫҢШҜ ШҙШҜ ЩҲ ШӘШұЩҒЩҶШҜЩҮШ§ЫҢ ШӯЫҢШ§ШӘЫҢ Щ…Ш§ЩҶЩҶШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ float64ШҢ ЩҶШұЩ…Ш§Щ„вҖҢШіШ§ШІЫҢШҢ ЩҲ ЩҲШІЩҶвҖҢШҜЩҮЫҢ ЩҫЩҲЫҢШ§ ШұШ§ ЫҢШ§ШҜ Ш®ЩҲШ§ЩҮЫҢШҜ ЪҜШұЩҒШӘ.

ШӘЩ…ШұЫҢЩҶвҖҢЩҮШ§ЫҢ Ш№Щ…Щ„ЫҢ

  1. ШӘЩ…ШұЫҢЩҶ Ыұ (ШӘШЈШ«ЫҢШұ ШіШ·Шӯ ЩҶЩҲЫҢШІ ШЁШұ ШӘШ®Щ…ЫҢЩҶ): Ъ©ШҜ ШұШ§ ШЁШ§ ШіШ·ЩҲШӯ ЩҶЩҲЫҢШІ Ыұ\%ШҢ Ыө\%ШҢ ЩҲ ЫұЫ°\% Ш§Ш¬ШұШ§ Ъ©ЩҶЫҢШҜ ЩҲ Ш®Ш·Ш§ЫҢ L2 ШұШ§ Щ…ЩӮШ§ЫҢШіЩҮ Ъ©ЩҶЫҢШҜ.
  2. ШӘЩ…ШұЫҢЩҶ ЫІ (ШӘШЈШ«ЫҢШұ ШӘШ№ШҜШ§ШҜ ШҜШ§ШҜЩҮвҖҢЩҮШ§): ШӘШ№ШҜШ§ШҜ ЩҶЩӮШ§Ш· ШҜШ§ШҜЩҮ ШұШ§ ШЁЩҮ ЫөЫ° ЩҲ ЫөЫ°Ы° ШӘШәЫҢЫҢШұ ШҜЩҮЫҢШҜ ЩҲ ЩҶШӘШ§ЫҢШ¬ ШұШ§ Щ…ЩӮШ§ЫҢШіЩҮ Ъ©ЩҶЫҢШҜ.
  3. ШӘЩ…ШұЫҢЩҶ Ыі (ШіЫҢШіШӘЩ… ШЁШ§ Щ…ЫҢШұШ§ЫҢЫҢ Ъ©Щ…): Ш¶ШұЫҢШЁ Щ…ЫҢШұШ§ЫҢЫҢ ШұШ§ ШЁЩҮ $c_{true} = 0.05$ ШӘШәЫҢЫҢШұ ШҜЩҮЫҢШҜ.
  4. ШӘЩ…ШұЫҢЩҶ Ыҙ (ШӘШұЪ©ЫҢШЁ ШЁШ§ ШҙЩҶШ§ШіШ§ЫҢЫҢ ЩҫШ§ШұШ§Щ…ШӘШұ): ЩҒШұШ¶ Ъ©ЩҶЫҢШҜ $c_{true}$ ЩҲ $k_{true}$ Щ…Ш¬ЩҮЩҲЩ„ ЩҮШіШӘЩҶШҜ ЩҲ ШўЩҶЩҮШ§ ШұШ§ ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ nn.Parameter ШӘШ№ШұЫҢЩҒ Ъ©ЩҶЫҢШҜ.
